# AI Resume Analyzer using MCP

## Overview

This project analyzes resume text and provides:

- ATS Score
- Skill Extraction
- Resume Feedback

Built using:

- Python
- MCP (Model Context Protocol)

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Run

```bash
python client.py
```

Paste resume text when prompted.

---

## Example

Input:

Python
SQL
Machine Learning
React

Output:

ATS Score: 40

Skills Found:
- python
- sql
- machine learning
- react

Feedback:
Needs improvement. Include more technical skills.
